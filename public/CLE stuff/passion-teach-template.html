Welcome to Basic HTML

HTML is the simple language used to build all websites. The browser reads it from top to bottom and displays the content.

1. What HTML Looks Like

HTML uses tags to show what things are. Tags usually come in pairs.

<tagname>This is content</tagname>
Example:
<p>This is a paragraph.</p>

2. The Basic Page Structure

Every page starts with a simple layout like this:

<!DOCTYPE html>
<html>
  <head>
    <title>My Page</title>
  </head>
  <body>
    Page content goes here!
  </body>
</html>

The <body> is the part the viewer sees.

Browser Output:
Page content goes here!

3. Headings

Headings come in sizes from 1 (biggest) to 6 (smallest).

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
...
Browser Output:

Heading 1

Heading 2

Heading 3

4. Paragraphs

The <p> tag is used for normal text.

<p>This is a simple paragraph.</p>
Browser Output:

This is a simple paragraph.

5. Images

You can show an image using the <img> tag. It does not need a closing tag.

<img src="example.jpg" width="200">
Browser Output:
(If example.jpg existed, it would appear here.)

6. Links

Links are created with the <a> tag.

<a href="https://example.com">Click here</a>
Browser Output:
Click here

7. Changing Text with JavaScript

JavaScript can change the content of your page. You can use getElementById to grab an element and then change its text.

// Grab the element by its ID let myText = document.getElementById("demo"); // Change its content myText.innerHTML = "Hello, JavaScript!";
Example:

This text will change.

Here, when you click the button, the paragraph text changes.
